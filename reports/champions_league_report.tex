\section{Introduction}
This project aims to build a comprehensive dataset of UEFA Champions League matches from 2011 to 2025. The dataset includes technical statistics, tactical lineups, and official match details to enable advanced sports performance analytics.

\section{Data Engineering Pipeline}
The pipeline is designed to be incremental and robust, merging data from multiple official and semi-official sources.

\subsection{Sources}
\begin{itemize}
    \item \textbf{UEFA Official API:} Used for match officials, kickoff times, and official lineups.
    \item \textbf{ESPN Summary API:} Used for granular match statistics (possession, shots, fouls) and event timelines.
    \item \textbf{Manual Overrides:} A JSON-based cache handles edge cases where automated discovery fails due to team name variations or historical discrepancies.
\end{itemize}

\subsection{Data Fields}
The following fields are extracted and validated:
\begin{itemize}
    \item \textbf{Structural:} Season, fase, jornada, fecha, local, visitante, resultado, etc.
    \item \textbf{Officials:} Principal referee and linesmen.
    \item \textbf{Tactical:} Starting lineups (planteles) and coaches.
    \item \textbf{Statistical:} Total shots, shots on target, possession \%, fouls, and corners.
\end{itemize}

\section{Conclusion}
The ingestion pipeline is fully operational and has successfully processed over 400 historical matches. While most statistical fields are complete, assist data remains a challenge due to API limitations and will be addressed in future milestones using alternative scraping techniques.
